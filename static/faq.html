Everything you need to know about detecting hidden text in assignment files

What is an AI assignment checker?
An AI assignment checker is a tool that scans PDF and DOCX assignment files to detect hidden text, invisible characters, and white text that teachers embed to catch AI users. When you upload an assignment file to ChatGPT, Claude, or other AI agents, these AI systems can read hidden content that's invisible to human eyes. FaireFile shows you exactly what AI agents will see in your file before you submit it, helping you avoid false accusations of AI cheating.
Why should I check my assignment file before submitting to AI agents like ChatGPT?
Many teachers are embedding invisible instructions in assignment files that only AI can read. These hidden instructions might include phrases like "mention Frankenstein" or "reference Batman" that appear nowhere in the visible assignment text. When you submit the file to ChatGPT or other AI agents, the AI reads these hidden references and includes them in its output. Your teacher then sees these references in your work and accuses you of AI cheating, even if you had no idea the hidden text existed. Checking your file with FaireFile before using AI prevents these false accusations.
What hidden content does FaireFile detect?
FaireFile detects multiple types of hidden content:
  • White text: Text colored white (RGB 255,255,255) that blends with the background
  • Invisible Unicode characters: Zero-width spaces, zero-width joiners, and other invisible characters (U+200B, U+200C, U+200D, etc.)
  • Microscopic text: Text with font size under 2pt that's nearly invisible
  • Hidden text properties: In Word documents, text marked with the "hidden" formatting property
  • Control characters: Various Unicode control and formatting characters used to embed invisible instructions
  • Tag characters: Unicode tag characters (U+E0000 to U+E007F) used for invisible watermarks

FaireFile currently supports PDF and DOCX (Microsoft Word) files, which are the most common formats used for academic assignments. How accurate is FaireFile at detecting hidden text?
FaireFile uses advanced text parsing and character analysis to detect hidden content with high accuracy. Our scanner examines PDF files at the character level to identify white text, analyzes Unicode character codes to find invisible characters, and checks font sizes to detect microscopic text. For DOCX files, we analyze formatting properties, text colors, and hidden text flags. While no tool is 100% perfect, FaireFile detects the most common methods teachers use to embed hidden instructions that AI agents can read.
